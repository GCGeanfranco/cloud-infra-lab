# cloud-infra-lab
Hands-on Cloud Engineering lab: AWS networking, Linux hardening, IaC with Terraform and CI/CD pipeline implementation.

## Objective
Design and implement a production-style AWS infrastructure using best practices in networking, security and automation.

## Stack
- AWS (VPC, EC2, RDS, ALB, IAM, CloudWatch)
- Linux (Hardening, SSH, Firewall)
- Terraform (Infrastructure as Code)
- GitHub Actions (CI/CD)
